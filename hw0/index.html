The doctor program uses elementary pattern matching on user input to produce a string as response to the user. Various keywords in the user input are used for constructing these replies so as to maintain the context. It has been fed with some knowledge like brother, mother, father belong to family and other similar facts. It has been fed with knowledge of various parts of a sentence like
  • Adjectives
  • Adverbs
  • Articles
  • Prepositions
It has also been fed with the knowledge of numbers, colors, sizes (like small, big, large etc), vowels. It has been made to understand drugs and alcohol, and various other things related to normal life, doctors. A set of random adjectives and other such forms are also have been fed.

Whenever the doctor fails to understand user input, it tries to continue the conversation by giving a very general response or by pointing to something said by the user in the past.
